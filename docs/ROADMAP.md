| 기능 | 상태 | 설명 |
|------|------|------|
| 온보딩 투자성향 테스트 | ✅ | 6개 질문, 안정형/중립형/공격형 판별, 맞춤 학습 경로 |
| 나만의 단어장 | ✅ | 용어 북마크 저장, 플래시카드 복습, 암기 완료 토글 |
| 시장 심리 대시보드 | ✅ | 공포/탐욕 SVG 게이지, 7일 추이, 뉴스피드, 종목별 감성 |

| 학습 마라톤 | ✅ | 17개 체크포인트, 세로 타임라인, 완주율 추적 |
| 신규 배지 4개 | ✅ | 투자 탐색가, 단어 수집가, 하프/풀 마라톤 |
| 커뮤니티 게시판 | 질문/답변, 종목 토론 |

**기술 고려사항**:
- Supabase: PostgreSQL + Auth + Realtime
- Claude API: 학습 추천 및 질문 답변
- 게이미피케이션: Zustand + Supabase 연동

## 차별화 포인트

1. **게이미피케이션**: 레벨, 배지, 스트릭으로 학습 동기부여
2. **인터랙티브 학습**: 읽기만 하는 게 아니라 직접 해보는 체험형
3. **실전 연계**: 이론 → 퀴즈 → 차트실습 → 모의투자
4. **한국+미국 시장**: 양쪽 시장 비교 학습
5. **AI 맞춤 학습**: 약점 분석 및 추천 (Phase 5)
